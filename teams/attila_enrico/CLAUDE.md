# Personal Finance App
This is a personal financial web app that helps users track their spendings and incomes. The users will register to this app by giving their e-mail addresses and passwords.

The web app should simple, easy to use and designed beautifully. 

#  Listing the transactions

Customers can list their transaction with filtering and sorting.

# Graphs and charts

Customers can see their spendings and incomes on various charts and graphs.

# User management

During registration / accoount creation the user has to pick a currency.

# Data input

Financial data
* Title / Description
* Amount (without currency)
* Timestamp
* Category: 
 - Spendings: Electircity bill, Movie ticket, 
 - Incomes: Salary, Occasional sell
