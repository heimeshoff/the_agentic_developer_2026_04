# Glossary

Accepted ubiquitous language only. Entries land here only after a human accepts a `## Glossary Proposals` candidate emitted by a skill.

**Rules**:

- Skills propose; humans accept. Never auto-edit this file.
- Existing entries are not silently overwritten. Updates are proposed the same way: "term X currently defined as Y; conversation suggests Z; update?".
- Each term is bound to a context. The same word may have a different definition in a different bounded context — record both, do not collapse them.

**Format**:

```markdown
### [Term]
- **Context**: [bounded context name, or "global" if domain-wide]
- **Definition**: [precise, one or two sentences]
- **Source**: [artifact path or conversation date]
```
